## 一、核心接口定义

### 1.1 AIPlugin 主接口

```typescript
/**
 * AI插件统一接口
 * 所有AI功能通过此接口暴露，功能模块仅依赖此接口，不直接调用任何AI实现
 */
interface AIPlugin {
  /** 检测AI服务是否可用（网络连通性+服务端健康检查） */
  isAvailable(): boolean

  /** 获取当前AI服务的状态详情 */
  getStatus(): AIStatus

  /** 笔记智能摘要 */
  summarize(text: string, options?: SummarizeOptions): Promise<string>

  /** 从笔记内容自动生成闪卡 */
  generateCards(note: string, options?: GenerateCardsOptions): Promise<GeneratedFlashcard[]>

  /** 评估费曼讲解质量 */
  evaluateExplanation(text: string, concept: string): Promise<EvaluationResult>

  /** 基于历史数据推荐番茄钟时长配置 */
  recommendDuration(history: PomodoroHistory[]): Promise<DurationConfig>
}
```

### 1.2 类型定义

```typescript
// ─── AI状态 ────────────────────────────────────────────────────────────────

interface AIStatus {
  available: boolean
  reason?: 'online' | 'offline' | 'rate_limited' | 'server_error' | 'not_loaded'
  latency?: number        // 最近一次请求延迟（ms）
  quotaRemaining?: number // 今日剩余配额
  model?: string          // 当前使用的模型名称
}

// ─── 笔记摘要 ───────────────────────────────────────────────────────────────

interface SummarizeOptions {
  maxLength?: number       // 摘要最大字数（默认200）
  style?: 'bullet' | 'paragraph' | 'outline'  // 摘要风格
  language?: 'zh' | 'en'  // 输出语言（默认跟随输入）
  focusKeywords?: string[] // 重点关注关键词
}

// ─── 闪卡生成 ───────────────────────────────────────────────────────────────

interface GenerateCardsOptions {
  maxCards?: number        // 最多生成卡片数（默认10）
  difficulty?: 'easy' | 'medium' | 'hard'  // 难度偏好
  cardType?: 'definition' | 'question_answer' | 'fill_blank' | 'mixed'
  language?: 'zh' | 'en'
}

interface GeneratedFlashcard {
  front: string            // 卡片正面
  back: string             // 卡片背面
  type: 'definition' | 'question_answer' | 'fill_blank'
  confidence: number       // AI生成置信度（0-1）
  sourceText?: string      // 来源文本片段
}

// ─── 费曼评估 ───────────────────────────────────────────────────────────────

interface EvaluationResult {
  overallScore: number     // 总体评分（1-10）
  clarity: number          // 清晰度（1-10）
  completeness: number     // 完整性（1-10）
  accuracy: number         // 准确性（1-10）
  strengths: string[]      // 讲解优点
  weaknesses: string[]     // 薄弱点/遗漏点
  suggestions: string[]    // 改进建议
  simplifiedVersion?: string // AI简化版讲解（供参考）
}

// ─── 番茄钟推荐 ─────────────────────────────────────────────────────────────

interface PomodoroHistory {
  date: string             // 日期 (YYYY-MM-DD)
  focusDuration: number    // 专注时长（分钟）
  completed: boolean       // 是否完成
  timeOfDay: 'morning' | 'afternoon' | 'evening' | 'night'
}

interface DurationConfig {
  focusMinutes: number     // 推荐专注时长（分钟）
  shortBreakMinutes: number
  longBreakMinutes: number
  cyclesBeforeLongBreak: number
  reason: string           // 推荐理由
  confidence: number       // 推荐置信度（0-1）
}
```

## 二、各AI功能的详细接口

### 2.1 笔记摘要 API

**功能描述：** 对笔记全文或选中段落生成结构化摘要，支持多种输出风格。

**调用流程：**
```
用户在笔记模块选中"生成摘要"
    ↓
NoteModule.summarizeNote(noteId)
    ↓
获取笔记内容 → 调用 AIPlugin.summarize(content, options)
    ↓
├─ AI可用：发送HTTPS请求到AI服务网关
│   ↓
│   FastAPI → gRPC调用LangChain Chain → LLM
│   ↓
│   返回摘要文本
└─ AI不可用：抛出 AINotAvailableError
    ↓
渲染摘要到笔记侧边栏
```

**服务端API：**
```
POST /api/ai/summarize
Authorization: Bearer <token>
Content-Type: application/json

{
  "text": "笔记全文内容...",
  "options": {
    "maxLength": 200,
    "style": "bullet",
    "language": "zh"
  }
}

// 响应
{
  "summary": "• 要点一：...\n• 要点二：...\n• 要点三：...",
  "model": "gpt-4o-mini",
  "tokensUsed": 350,
  "latencyMs": 1200
}
```

**Prompt模板：**
```
你是一个学习笔记摘要助手。请对以下笔记内容生成结构化摘要。

要求：
- 风格：{style}
- 字数不超过{maxLength}字
- 提取核心概念、关键定义、重要关系
- 使用学生能理解的语言

笔记内容：
{text}

摘要：
```

### 2.4 番茄钟推荐 API

**功能描述：** 基于用户的历史专注数据，推荐最优的番茄钟时长配置。

**调用流程：**
```
用户在番茄钟设置页点击"智能推荐"
    ↓
获取最近30天番茄钟历史数据
    ↓
AIPlugin.recommendDuration(history)
    ↓
├─ AI可用：
│   FastAPI → 分析历史模式 → 生成推荐
│   ↓
│   返回 DurationConfig
└─ AI不可用：
    使用本地规则引擎推荐（基于统计的简单算法）
    ↓
UI展示推荐配置+推荐理由
    ↓
用户确认应用 or 手动调整
```

**本地降级规则引擎（AI不可用时）：**
```typescript
class LocalDurationRecommender {
  recommend(history: PomodoroHistory[]): DurationConfig {
    const completedSessions = history.filter(h => h.completed);
    const avgDuration = completedSessions.reduce((sum, h) => sum + h.focusDuration, 0)
      / Math.max(completedSessions.length, 1);

    // 基于平均完成时长推荐
    let focusMinutes: number;
    if (avgDuration >= 45) focusMinutes = 50;
    else if (avgDuration >= 30) focusMinutes = 35;
    else if (avgDuration >= 20) focusMinutes = 25;
    else focusMinutes = 20;

    // 基于时段分析
    const morningCompletion = this.getCompletionRate(history, 'morning');
    const eveningCompletion = this.getCompletionRate(history, 'evening');

    return {
      focusMinutes,
      shortBreakMinutes: Math.round(focusMinutes * 0.2),
      longBreakMinutes: Math.round(focusMinutes * 0.4),
      cyclesBeforeLongBreak: 4,
      reason: `基于你最近${history.length}次记录，平均专注${Math.round(avgDuration)}分钟，推荐${focusMinutes}分钟专注时长`,
      confidence: Math.min(completedSessions.length / 30, 1),
    };
  }
}
```

## 三、降级策略设计

### 3.1 核心原则

> AI功能按钮始终显示在UI中，不因AI不可用而隐藏或禁用任何按钮。

### 3.2 降级流程

```
用户点击AI功能按钮
    ↓
检测 AIPlugin.isAvailable()
├─ true → 正常调用AI服务
│   ├─ 调用成功 → 渲染结果
│   └─ 调用失败（超时/服务端错误）
│       → 显示错误提示"AI服务暂时不可用，请稍后重试"
│       → 提供"重试"按钮
│
└─ false → 触发降级处理
    │
    ├─ 笔记摘要：显示"连接网络后可使用智能摘要功能"
    │            + 提供"复制全文"快捷操作
    │
    ├─ 闪卡生成：显示"连接网络后可使用智能生成闪卡功能"
    │            + 提供"手动创建闪卡"入口
    │
    ├─ 费曼评估：显示"连接网络后可使用AI评估功能"
    │            + 提供"自我评分"替代方案（1-5星评分）
    │
    └─ 番茄钟推荐：自动降级到本地规则引擎
                    显示"基于本地数据的推荐"（不提示AI不可用）
```

### 3.3 降级策略对比表

| AI功能 | 降级方式 | 替代方案 | 用户感知 |
|--------|---------|---------|---------|
| 笔记摘要 | 友好提示 | 手动选择+复制 | 明确知道需联网 |
| 闪卡生成 | 友好提示 | 手动创建入口 | 明确知道需联网 |
| 费曼评估 | 友好提示 | 自我评分（1-5星） | 有基础替代方案 |
| 番茄钟推荐 | 无感降级 | 本地规则引擎 | 完全无感知 |

### 3.4 不使用Feature Flag的原因

**决策：不通过Feature Flag隐藏AI功能。**

理由：
1. **用户期望一致**：所有用户看到相同的UI，知道课伴具备AI能力
2. **功能发现**：离线用户看到AI按钮后，联网时会主动尝试
3. **教育价值**：提示用户"需要网络"比"隐藏功能"更透明
4. **营销价值**：AI功能是课伴的差异化卖点，不应隐藏

### 4.4 gRPC Protobuf接口定义

> v1.1新增：AI服务与同步服务之间通过gRPC通信，以下为核心接口定义。

```protobuf
// ai_service.proto
syntax = "proto3";
package keban.ai.v1;

service AIService {
  // 笔记摘要

message SummarizeRequest {
  string user_id = 1;
  string text = 2;
  SummarizeOptions options = 3;
}

message SummarizeOptions {
  int32 max_length = 1;
  string style = 2;      // bullet | paragraph | outline
  string language = 3;
}

message SummarizeResponse {
  string summary = 1;
  string model = 2;
  int32 tokens_used = 3;
  int32 latency_ms = 4;
}

message GenerateCardsRequest {
  string user_id = 1;
  string note_content = 2;
  GenerateCardsOptions options = 3;
}

message GenerateCardsOptions {
  int32 max_cards = 1;
  string difficulty = 2;
  string card_type = 3;
}

message GenerateCardsResponse {
  repeated GeneratedCard cards = 1;
  int32 total_extracted = 2;
  string model = 3;
  int32 tokens_used = 4;
}

  string front = 1;
  string back = 2;
  string type = 3;

  string source_text = 5;
}

  string user_id = 1;
  string concept = 2;
  string explanation = 3;
}

  int32 overall_score = 1;
  int32 clarity = 2;
  int32 completeness = 3;
  int32 accuracy = 4;
  repeated string strengths = 5;
  repeated string weaknesses = 6;
  repeated string suggestions = 7;

}

message SummarizeChunk {
  string text = 1;
  bool is_final = 2;
  int32 progress_percent = 3;
}
```

## 五、错误处理与超时策略

### 5.1 错误分类与处理

```typescript
// 错误类型层级
class AIError extends Error {
  constructor(message: string, public readonly type: AIErrorType) {
    super(message);
  }
}

type AIErrorType =
  | 'not_available'    // AI服务不可用（离线/未加载）
  | 'timeout'          // 请求超时
  | 'rate_limited'     // 调用频率超限
  | 'server_error'     // 服务端内部错误
  | 'invalid_input'    // 输入内容无效（过短/格式错误）
  | 'quota_exceeded';  // 用户配额耗尽

// 客户端错误处理
async function callAIWithHandling<T>(
  fn: () => Promise<T>,
  fallback: T | (() => T)
): Promise<T> {
  try {
    return await fn();
  } catch (error) {
    if (error instanceof AIError) {
      switch (error.type) {
        case 'not_available':
          showFriendlyMessage('连接网络后可使用智能功能');
          return typeof fallback === 'function' ? fallback() : fallback;

        case 'timeout':
          showToast('AI响应超时，请稍后重试', { action: '重试', onAction: fn });
          throw error;

        case 'rate_limited':
          showFriendlyMessage('今日AI使用次数已达上限，明天再来吧');
          return typeof fallback === 'function' ? fallback() : fallback;

        case 'server_error':
          showToast('AI服务暂时不可用，请稍后重试');
          throw error;

        case 'invalid_input':
          showToast('内容太短，无法生成有效结果');
          throw error;

        case 'quota_exceeded':
          showUpgradePrompt();
          throw error;
      }
    }
    throw error;
  }
}
```

### 5.3 超时处理实现

```typescript
async function withTimeout<T>(
  promise: Promise<T>,
  timeoutMs: number,
  operation: string
): Promise<T> {
  const controller = new AbortController();

  const timeoutId = setTimeout(() => {
    controller.abort();
  }, timeoutMs);

  try {
    const result = await Promise.race([
      promise,
      new Promise<never>((_, reject) =>
        controller.signal.addEventListener('abort', () =>
          reject(new AIError(`${operation}超时(${timeoutMs}ms)`, 'timeout'))
        )
      ),
    ]);
    clearTimeout(timeoutId);
    return result;
  } catch (error) {
    clearTimeout(timeoutId);
    throw error;
  }
}
```

```python
from fastapi import Request, HTTPException
from redis.asyncio import Redis
import time

class RateLimiter:
    def __init__(self, redis: Redis):
        self.redis = redis

    async def check(self, user_id: str, feature: str, limits: dict) -> bool:
        today = time.strftime('%Y-%m-%d')

        # 检查功能级限制
        feature_key = f"ratelimit:{user_id}:{feature}:{today}"
        feature_count = await self.redis.get(feature_key)
        if feature_count and int(feature_count) >= limits.get(feature, 10):
            return False

        # 检查总限制
        total_key = f"ratelimit:{user_id}:total:{today}"
        total_count = await self.redis.get(total_key)
        if total_count and int(total_count) >= limits.get('daily_total', 50):
            return False

        # 计数+1
        await self.redis.incr(feature_key)
        await self.redis.expire(feature_key, 86400)
        await self.redis.incr(total_key)
        await self.redis.expire(total_key, 86400)

        return True

# FastAPI依赖
async def rate_limit_dependency(
    request: Request,
    redis: Redis = Depends(get_redis),
    user: User = Depends(get_current_user),
):
    limiter = RateLimiter(redis)
    feature = request.url.path.split('/')[-1]  # e.g., "summarize"

    allowed = await limiter.check(user.id, feature, RATE_LIMITS)
    if not allowed:
        raise HTTPException(
            status_code=429,
            detail={"error": "rate_limited", "message": "今日使用次数已达上限"}
        )
```

### 6.4 结果缓存策略

```python
import hashlib
from redis.asyncio import Redis

class ResultCache:
    def __init__(self, redis: Redis):
        self.redis = redis

    async def get_or_compute(self, key_parts: list[str], compute_fn, ttl: int = 604800):
        """获取缓存结果或执行计算并缓存"""
        # 生成缓存键（基于输入内容的SHA256）
        content = '|'.join(key_parts)
        cache_key = f"ai:cache:{hashlib.sha256(content.encode()).hexdigest()}"

        # 尝试从缓存获取
        cached = await self.redis.get(cache_key)
        if cached:
            return json.loads(cached)

        # 缓存未命中，执行计算
        result = await compute_fn()

        # 缓存结果（默认7天）
        await self.redis.setex(cache_key, ttl, json.dumps(result))

        return result

# 使用示例
async def summarize_with_cache(text: str, options: dict, cache: ResultCache):
    return await cache.get_or_compute(
        key_parts=[text, json.dumps(options, sort_keys=True)],
        compute_fn=lambda: call_llm_summarize(text, options),
        ttl=604800  # 7天缓存
```

## 附录：AI插件加载流程

```typescript
// AI插件动态加载器
class AIPluginLoader {
  private plugin: AIPlugin | null = null;
  private loading: Promise<AIPlugin> | null = null;

  async getPlugin(): Promise<AIPlugin> {
    // 已加载，直接返回
    if (this.plugin?.isAvailable()) return this.plugin;

    // 正在加载，返回同一个Promise
    if (this.loading) return this.loading;

    // 开始加载
    this.loading = this.loadPlugin();
    try {
      this.plugin = await this.loading;
      return this.plugin;
    } catch {
      // 加载失败，返回LocalFallback
      this.plugin = new LocalFallback();
      return this.plugin;
    } finally {
      this.loading = null;
    }
  }

  private async loadPlugin(): Promise<AIPlugin> {
    // 动态import，仅在首次使用时加载
    const { RemoteAIPlugin } = await import('./plugins/remote-ai-plugin');
    const plugin = new RemoteAIPlugin({
      baseUrl: import.meta.env.VITE_AI_GATEWAY_URL,
      getToken: () => authStore.getToken(),
    });

    // 验证服务可用性
    if (!plugin.isAvailable()) {
      throw new Error('AI service not available');
    }

    return plugin;
  }
}
```
