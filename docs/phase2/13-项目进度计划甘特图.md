## 八、进度监控机制

### 8.1 周报制度

每周五提交进度周报，包含：

- 本周完成的任务（与计划对比）
- 下周计划任务
- 延期/风险项说明
- 缓冲消耗情况
- SPI（进度绩效指数）计算

### 8.2 进度偏差预警

| SPI 范围 | 状态 | 处理方式 |
|---------|------|---------|
| SPI ≥ 1.0 | 正常（绿色） | 按计划推进 |
| 0.9 ≤ SPI < 1.0 | 轻微偏差（黄色） | 加强每日站会跟踪 |
| 0.8 ≤ SPI < 0.9 | 显著偏差（橙色） | 分析原因，调整资源 |
| SPI < 0.8 | 严重偏差（红色） | 启动应急预案，考虑范围裁剪 |

### 8.3 里程碑检查点

每个里程碑到达时进行正式评审：

```
检查点评审流程：
  1. 交付物完整性检查
  2. 质量标准达标验证
  3. 进度偏差分析
  4. 下一阶段计划确认
  5. 风险登记册更新
  6. 评审会议纪要归档
```
